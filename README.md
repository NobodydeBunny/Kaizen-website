<h1> ${\color{purple}Kaizen \space Website}$</h1>

Kaizen-website is a lightweight, student-level static website built as a mini project for university. It showcases basic web design with HTML, CSS, and JavaScript, and includes simple interactive features and visual components.

This repository contains all the source files for the site, including layout files, styles, scripts, and assets. It is suitable for learning how a static website is structured and how to present content effectively.

<h2>Project Description</h2>

This website was created to demonstrate core web development fundamentals. It blends static HTML pages with styled CSS and basic interactive JavaScript. Its design focuses on clarity, simplicity, and responsiveness.

<h3>Key features include:</h3>
<ul>
<li>Clean semantic HTML structure</li>

<li>Organized CSS for layout and style</li>

<li>JavaScript for any basic interactivity</li>

<li>Content aimed at practical web usage</li>
</ul>
